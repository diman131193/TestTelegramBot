## Текущее состояние

Проект реализует Telegram-бота на `aiogram 3.29.1` для мастера по реконструкции и восстановлению волос. Бот показывает стартовое меню, клиентские услуги, отзывы, консультацию через администратора и бесплатную диагностику волос.

В спеках отдельно отмечены обнаруженные разрывы текущей реализации:

- кнопка `Прайс` существует в клавиатурах, но для callback `price` нет обработчика;
- текстовый ключ `price` отсутствует в `data/texts.json`;
- диагностика собирает один основной вывод, стоп-флаги и контекстные дополнения;
- контент редактируется в `content/diagnostics-content.xlsx` и проверяется по всем 3 696 путям до публикации;
- прогресс диагностики сохраняется в SQLite и переживает перезапуск процесса;
- состояние консультации пока хранится только в памяти процесса.
